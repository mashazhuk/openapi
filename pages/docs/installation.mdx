---
title: Installation
---


To run the social network project locally, follow these steps:

```bash
# 1. Clone the repository
$ git clone https://github.com/mashazhuk/social-network.git
$ cd social-network

# 2. Install dependencies
$ npm install

# 3. Setup environment variables
$ cp .env.example .env
# Edit the .env file with your database credentials

# 4. Run the development server
$ npm run dev

# 5. Access the app at http://localhost:3000
```

Backend should be set up separately with appropriate API endpoints for user and post management.
